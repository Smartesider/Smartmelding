SmartMelding lar deg sende helsemeldinger, AI-genererte varsler og PDF-rapporter fra WordPress.

== Description ==
SmartMelding er en fleksibel plugin utviklet for å sende og arkivere meldinger og helseinformasjon, generere PDF-er og bruke AI-moduler for tekst og innhold. 
Støtter cron-jobber, REST API, oversettelser og shortcodes.

== Installation ==
1. Last opp plugin-mappen til `/wp-content/plugins/`
2. Aktiver pluginen via `Plugins`-menyen i WordPress
3. Gå til Innstillinger -> SmartMelding for å konfigurere

== Frequently Asked Questions ==
= Trenger jeg en AI-nøkkel? =
Ja, for full AI-funksjonalitet må du legge inn din OpenAI-nøkkel under Innstillinger.
